# 📖 Sobre o Projeto

O **FastQuest** é uma plataforma digital voltada para a preparação de estudantes de Direito para o **Exame da Ordem dos Advogados do Brasil (OAB)**.

A aplicação centraliza um banco de questões e permite que alunos realizem **simulados personalizados**, acompanhem seu desempenho e identifiquem seus pontos fracos em cada disciplina.

Além disso, professores podem criar listas de exercícios e analisar o progresso das turmas de forma automatizada.

Nosso objetivo é transformar a forma como estudantes se preparam para a OAB, substituindo métodos manuais por uma experiência digital inteligente e eficiente.

---

# ✨ Funcionalidades Principais

📚 **Banco de Questões**
- Cadastro e organização de questões por disciplina e tema

📝 **Simulados Personalizados**
- Geração automática de simulados por disciplina ou ano da prova

📊 **Análise de Desempenho**
- Estatísticas de acertos por matéria

🔍 **Busca Avançada**
- Filtragem por disciplina, tema ou prova específica

🧠 **Histórico de Resolução**
- Registro das respostas dos usuários ao longo do tempo

# 🏗️ Arquitetura do Sistema

O FastQuest utiliza uma arquitetura **web baseada em API REST**.

Frontend (Vue.js)
↓
API REST (Golang)
↓
Banco de Dados (PostgreSQL)


### Componentes

**Frontend**
- Interface do usuário
- Renderização dos simulados
- Dashboard de desempenho

**Backend**
- Regras de negócio
- Processamento de respostas
- Geração de simulados

**Banco de Dados**
- Armazenamento de questões
- Histórico de respostas
- Dados de usuários
